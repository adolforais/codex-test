How to read this

The drawdown shows how far the ETF is from its record close. A smaller negative value means the price is near the high; a positive value indicates a new high.

Data pulls use the Alpha Vantage Time Series API. When the API key is missing or rate-limited, the dashboard falls back to bundled sample data.
